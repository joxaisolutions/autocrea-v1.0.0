  1. Asegúrate que el servidor está corriendo:
     cd frontend && npm run dev

  2. Ve al Dashboard:
     http://localhost:3000/dashboard

  3. Click en "Try Editor" en Quick Actions

  4. Selecciona un archivo del árbol (izquierda)

  5. Edita el código en Monaco Editor

  6. Prueba la terminal (parte inferior)
     - Escribe: help
     - Escribe: ls
     - Escribe: clear

  7. Navega entre tabs (Code/Preview)

  8. Prueba los botones del toolbar:
     - Save
     - Run
     - Share
     - Export
     - Toggle Terminal
